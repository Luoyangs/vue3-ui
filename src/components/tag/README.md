## API

**Tag props**
| Property | Description | Type | Accepted Values | Default |
|:--|:--|:--|:--|:--|
| type         | 标签类型 |      string       | 'success', 'info', 'warning', 'danger'   |   ''    |
| outline         | 标签是否带边框 |      boolean       | true/false   |   false   |
| closable         | 标签是否可以关闭 | boolean             | true/false  |    false   |
| color            | 标签色           | string              | -      |       |

**Tag event**
| 事件名称 | 说明         | 回调参数    |
| -------- | ------------ | ----------- |
| close    | 关闭时的回调 | (e) => void |
